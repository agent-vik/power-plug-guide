## 3. SEO优化策略

### 3.1 元数据优化

**标题优化（60字符内）**
```
原标题: World Power Plug Guide
优化后: World Power Plug Guide - 15 Types, Voltage & Travel Adapter Info
```

**描述优化（160字符内）**
```
原描述: Learn about plug types, voltage and frequency standards worldwide...
优化后: Complete guide to 15 plug types (A-O), global voltage standards, and travel adapters. Find what plug adapter you need for any country before your trip.
```

### 3.2 结构化数据（Schema.org）

添加以下结构化数据：
- WebSite
- FAQPage
- HowTo
- BreadcrumbList

### 3.3 内容SEO优化

#### H1标题优化
- 当前: "World Power Plug Guide"
- 优化: "World Power Plug Guide: Complete Travel Adapter Reference"

#### FAQ内容添加
```
Q: What plug adapter do I need for Japan?
A: Japan uses Type A and B plugs (same as US), but voltage is 100V...

Q: Do I need a voltage converter for Europe?
A: If your device supports 100-240V (check label), you only need a plug adapter...

Q: What's the difference between Type C and Type F plugs?
A: Type C (Europlug) has no ground, rated 2.5A. Type F (Schuko) has side grounding clips, rated 16A...
```

### 3.4 图片SEO

为每张插座图片添加：
- 描述性alt属性
- title属性
- 结构化数据中的ImageObject

### 3.5 内链优化

- 添加"相关国家"链接
- 添加"相关插座类型"链接
- 锚文本优化

### 3.6 多语言SEO

```
英文: /en/ 或默认
中文: /zh/ 或通过hreflang
```

添加hreflang标签：
```html
<link rel="alternate" hreflang="en" href="https://world-plugs.victor42.work/" />
<link rel="alternate" hreflang="zh" href="https://world-plugs.victor42.work/?lang=zh" />
```
