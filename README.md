# 全球电源插座指南 | World Power Plug Guide

一个响应式单页科普网站，介绍全球主要电源插座标准、电压和频率。

## 项目简介

本网站收录了全球15种主要电源插座类型（Type A ~ Type O），详细展示每种插座的：
- 高清插座图片
- 使用国家和地区
- 电压和频率标准
- 设计特点和历史背景

## 技术特点

- **纯静态网站** - 无需服务器，可直接部署到GitHub Pages
- **响应式设计** - 完美适配桌面端和移动端
- **WebP图片格式** - 所有插座图片使用WebP格式，加载速度快
- **搜索与筛选** - 支持按地区筛选和关键词搜索
- **SEO优化** - 完整的meta标签、Schema.org结构化数据、站点地图
- **零依赖** - 不使用任何外部框架，加载速度快
