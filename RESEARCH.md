# 全球电源插座标准研究报告

## 概述

本报告详细记录了全球电源插座标准的研究成果，作为 power-plug-guide 网站的事实依据。数据来源于国际电工委员会（IEC）、World Standards 等权威机构。

---

## 一、插座类型分类

世界上目前共有15种主要家用电源插座类型，由美国商务部以字母A至O命名。以下是各类型的详细规格。

### Type A（美标两扁插头）

**基本规格**
- 引脚数量：2个平行扁脚
- 引脚间距：12.7mm（0.5英寸）
- 引脚宽度：1.5mm
- 引脚厚度：6.4mm
- 额定电流：15A

**技术特点**
- 无接地设计，仅适用于I类绝缘设备
- 由Harvey Hubbell于1904年发明，是世界上最古老的插头设计之一
- 两个引脚通常带有小孔，用于与插座内的凸起配合防止脱落

**使用地区**
美国、加拿大、墨西哥、日本、台湾、泰国、越南、菲律宾、哥伦比亚、委内瑞拉等。

**相关标准**
- 美国：NEMA 1-15
- 日本：JIS C 8303

---

### Type B（美标三脚插头）

**基本规格**
- 引脚数量：2个平行扁脚 + 1个圆形接地引脚
- 扁脚间距：12.7mm
- 接地引脚直径：4.8mm
- 额定电流：15A（亦有20A规格）

**技术特点**
- 在Type A基础上增加了接地引脚
- 接地引脚比两个扁平引脚长约3mm，确保"先接地后通电"
- 接地引脚呈半圆形或U形，防止误插

**使用地区**
美国、加拿大、墨西哥、哥斯达黎加、危地马拉等北美及中美洲国家。

**相关标准**
- 美国：NEMA 5-15
- 加拿大：CSA C22.2 No.42

---

### Type C（欧标两脚圆插头 / Europlug）

**基本规格**
- 引脚数量：2个圆形引脚
- 引脚直径：4.0mm
- 引脚间距：19mm（中心距）
- 引脚长度：19mm
- 额定电流：2.5A

**技术特点**
- 也称为Europlug（欧洲插头）
- 设计精巧，引脚具有弹性，可适配多种欧洲插座（Type E、F、J、K、L等）
- 仅适用于不超过2.5A的小功率设备
- 无接地设计，仅适用于II类绝缘设备

**使用地区**
德国、法国、意大利、西班牙、荷兰、比利时、奥地利、瑞士、葡萄牙、希腊、土耳其、俄罗斯、波兰、捷克等欧洲大陆国家，以及部分亚洲和非洲国家。

**相关标准**
- 欧洲：CEE 7/16
- 国际：IEC 60884-1 C1a

---

### Type D（印度标三脚圆插头）

**基本规格**
- 引脚数量：3个圆形引脚，呈三角形排列
- 引脚直径：5.1mm（也有7.1mm大电流版本）
- 接地引脚位于顶部
- 额定电流：6A（标准），15A/16A（大电流版本）

**技术特点**
- 源自旧版英国标准BS 546（已被英国淘汰）
- 三个引脚间距较大，设计较为笨重
- 接地引脚比相线引脚略长

**使用地区**
印度、尼泊尔、巴基斯坦、斯里兰卡、孟加拉国、缅甸等南亚国家。

**相关标准**
- 印度：IS 1293
- 英国（已废止）：BS 546

---

### Type E（法标两脚圆插头）

**基本规格**
- 引脚数量：2个圆形引脚 + 插座侧接地引脚
- 引脚直径：4.8mm
- 引脚间距：19mm
- 额定电流：16A

**技术特点**
- 两个圆形引脚加上一个凸出的接地孔
- 插座有一个突出的接地引脚，插头有对应的接地孔
- 与Type F（Schuko）不兼容，因接地方式不同
- 接地引脚位于正上方

**使用地区**
法国、比利时、波兰、捷克、斯洛伐克、突尼斯、摩洛哥等。

**相关标准**
- 法国：NF C 61-314
- 欧洲：CEE 7/6（插头）、CEE 7/5（插座）

---

### Type F（德标两脚圆插头 / Schuko）

**基本规格**
- 引脚数量：2个圆形引脚 + 两侧接地夹
- 引脚直径：4.8mm
- 引脚间距：19mm
- 额定电流：16A

**技术特点**
- 名称Schuko来自德语"Schutzkontakt"（保护接触）
- 两侧的接地夹比引脚先接触，确保安全
- 是欧洲大陆最通用的接地插头标准
- 可兼容Type C（Europlug）插头

**使用地区**
德国、奥地利、荷兰、瑞典、挪威、芬兰、俄罗斯、乌克兰、葡萄牙、西班牙、希腊、土耳其等。

**相关标准**
- 德国：DIN VDE 0620
- 欧洲：CEE 7/4（插座）、CEE 7/7（插头）

---

### Type G（英标三脚方插头）

**基本规格**
- 引脚数量：3个长方形引脚，呈三角形排列
- 相线引脚：6.4mm × 4mm × 18mm
- 接地引脚：8mm × 4mm × 23mm（比相线长）
- 额定电流：13A（内置保险丝保护）

**技术特点**
- 接地引脚比相线引脚长，确保先接地
- 插头内部集成保险丝（通常为13A或更低），提供额外保护
- 插座带有安全百叶窗，防止儿童触电
- 设计精良，安全性极高，是世界上识别度最高的插头之一

**使用地区**
英国、爱尔兰、香港、新加坡、马来西亚、沙特阿拉伯、阿联酋、卡塔尔、塞浦路斯、马耳他等。

**相关标准**
- 英国：BS 1363
- 新加坡：SS 145
- 马来西亚：MS 589

---

### Type H（以色列标三脚插头）

**基本规格**
- 引脚数量：3个圆形引脚，呈Y形排列
- 引脚直径：4.5mm（旧版）或4.0mm（新版）
- 额定电流：16A

**技术特点**
- 以色列独特标准
- 现代版本也兼容Type C（Europlug）的扁脚
- Y形排列与三角形排列（Type D/M）不同

**使用地区**
以色列、巴勒斯坦、加沙地带。

**相关标准**
- 以色列：SI 32

---

### Type I（澳标三脚斜插头）

**基本规格**
- 引脚数量：2个斜向扁平引脚 + 1个垂直接地引脚
- 相线引脚：6.4mm × 1.6mm，呈V形排列（约30°倾斜）
- 接地引脚：垂直向下
- 额定电流：10A（亦有15A、20A规格）

**技术特点**
- 两个相线引脚呈V形排列，角度约为30°
- 中国使用的是简化版本（Type I的中国变体），两个扁脚平行排列
- 新西兰版本引脚稍细

**使用地区**
澳大利亚、新西兰、中国、阿根廷、巴布亚新几内亚、斐济、所罗门群岛等。

**相关标准**
- 澳大利亚/新西兰：AS/NZS 3112
- 中国：GB 2099.1、GB 1002
- 阿根廷：IRAM 2073

---

### Type J（瑞士标三脚插头）

**基本规格**
- 引脚数量：3个圆形引脚
- 引脚直径：4.0mm
- 引脚间距：19mm（相线引脚）
- 额定电流：10A

**技术特点**
- 与Type C类似但增加了接地引脚
- 插座为六边形设计，可以防止插错方向
- 引脚带有棱边，防止从多插头插座中滑出
- 可兼容Type C（Europlug）

**使用地区**
瑞士、列支敦士登、卢旺达。

**相关标准**
- 瑞士：SEV 1011

---

### Type K（丹麦标三脚插头）

**基本规格**
- 引脚数量：2个圆形引脚 + 1个U形接地引脚
- 引脚直径：4.8mm
- 引脚间距：19mm
- 额定电流：16A

**技术特点**
- 类似Type E，但接地引脚在插头上而非插座上
- 接地引脚呈U形或扁平状
- 丹麦现在也接受Type F（Schuko）插头

**使用地区**
丹麦、格陵兰、孟加拉国。

**相关标准**
- 丹麦：Afsnit 107-2-D1

---

### Type L（意大利标三脚圆插头）

**基本规格**
- 引脚数量：3个圆形引脚，排成一行
- 10A规格：引脚直径4.0mm，间距19mm
- 16A规格：引脚直径5.0mm，间距26mm
- 额定电流：10A或16A

**技术特点**
- 三个圆形引脚排成一行，独特设计
- 意大利有两种规格：10A（小间距）和16A（大间距）
- 现代意大利插座通常同时兼容两种规格
- 接地引脚位于中间

**使用地区**
意大利、智利、古巴、叙利亚、利比亚、马尔代夫、乌拉圭。

**相关标准**
- 意大利：CEI 23-16/VII

---

### Type M（南非标三脚圆插头）

**基本规格**
- 引脚数量：3个圆形引脚，呈三角形排列
- 引脚直径：7.1mm（比Type D更大）
- 额定电流：15A

**技术特点**
- 与Type D相似但引脚更大
- 是旧版英国标准BS 546的另一衍生版本（15A规格）
- 南非正在逐步过渡到Type N（IEC 60906-1标准）

**使用地区**
南非、斯威士兰、莱索托、博茨瓦纳、莫桑比克、中国香港（旧式建筑）、马来西亚（旧式）。

**相关标准**
- 南非：SANS 164-1
- 英国（已废止）：BS 546

---

### Type N（巴西标两脚圆插头）

**基本规格**
- 引脚数量：2个圆形引脚 + 接地孔
- 引脚直径：4.0mm
- 引脚间距：19mm
- 额定电流：10A或20A

**技术特点**
- 基于IEC 60906-1国际标准设计
- 两个圆形引脚加上接地孔（插座侧有接地引脚）
- 被认为是未来的国际统一标准候选
- 巴西各地电压不同，从127V到220V都有

**使用地区**
巴西、南非（新标准，正在过渡中）。

**相关标准**
- 国际：IEC 60906-1
- 巴西：NBR 14136
- 南非：SANS 164-2

---

### Type O（泰国三脚圆插头）

**基本规格**
- 引脚数量：2个圆形引脚 + 1个接地引脚
- 引脚直径：4.8mm
- 引脚间距：19mm
- 额定电流：16A

**技术特点**
- 泰国新标准，基于IEC 60906-1国际标准
- 正在逐步推广，与现有Type A/B/C插座并存
- 与Type N相似但规格略有不同

**使用地区**
泰国（新式建筑和酒店）。

**相关标准**
- 泰国：TIS 166-2549

## 二、电压与频率标准

### 全球电压分布

世界上主要存在两种电压标准体系：

| 电压范围 | 典型电压 | 使用地区 |
|----------|----------|----------|
| 100-127V | 110V, 115V, 120V, 127V | 北美、中美洲、日本、台湾、菲律宾部分、沙特部分 |
| 220-240V | 220V, 230V, 240V | 欧洲、亚洲、非洲、大洋洲、南美洲大部 |

**历史原因**
- 110V：爱迪生早期直流电系统的遗产，后被特斯拉交流电系统沿用
- 220V：欧洲在电力发展后期选择，更高的电压意味着更低的输电损耗
- 日本独特情况：东部（50Hz，东京电力）使用100V，西部（60Hz，大阪电力）使用100V

### 频率标准

| 频率 | 使用地区 | 历史背景 |
|------|----------|----------|
| 50Hz | 欧洲、亚洲、非洲、大洋洲、南美洲大部 | 德国AEG公司于1891年选择，后成为欧洲标准 |
| 60Hz | 北美、中美洲、日本西部、韩国、菲律宾、台湾 | 美国西屋公司选择，基于更高效的白炽灯设计 |

**特殊案例**
- 日本：全国100V，但东部50Hz，西部60Hz（历史遗留，两套电网无法直接互联）
- 韩国：220V 60Hz（与日本不同）
- 巴西：不同州电压不同（127V或220V），频率统一60Hz
- 沙特阿拉伯：正在从127V 60Hz过渡到230V 60Hz

## 四、旅行转换器与变压器指南

### 转换器 vs 变压器

| 类型 | 功能 | 适用场景 |
|------|------|----------|
| 转换插头 | 仅改变插头形状，不改变电压 | 设备支持宽电压（100-240V）时使用 |
| 变压器 | 改变电压（升压或降压） | 设备仅支持单一电压时使用 |

### 设备兼容性判断

查看设备铭牌或充电器上的标识：

| 标识示例 | 含义 | 使用建议 |
|----------|------|----------|
| Input: 100-240V~50/60Hz | 宽电压，全球通用 | 仅需转换插头 |
| Input: 110-120V~60Hz | 仅支持北美电压 | 去欧洲/亚洲需变压器 |
| Input: 220-240V~50Hz | 仅支持欧洲/亚洲电压 | 去北美/日本需变压器 |

### 功率匹配

使用变压器时，变压器额定功率应为设备功率的1.5倍以上：

- 吹风机（1000-2000W）→ 需要1500-3000W变压器
- 电熨斗（800-1500W）→ 需要1200-2250W变压器
- 笔记本电脑（45-100W）→ 需要70-150W变压器
